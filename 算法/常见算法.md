## 约瑟夫环

    100个人手拉手,数数,数到3的倍数的人退出,最后剩下的是之前的第几个人

``` java
public int solution(List<Integer> list ,int size,int div) {
   int point = 0;
   int number = 1;

   while (list.size() > 1) {
       if (number % div == 0) {
           list.remove(point);
           point--;
       }
       point++;
       number++;

       if (point > (list.size() - 1)) {
           point = 0;
       }
   }
   return list.get(0);
}
```
